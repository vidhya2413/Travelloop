# 🌍 TravelLoop - Travel Planning Web Application

TravelLoop is a simple and interactive travel planning web application that allows users to explore destinations, view travel packages, and book trips easily. It is built using HTML, CSS, Bootstrap, JavaScript, and basic PHP with MySQL database support.

---

## ✨ Features

- 🏠 Home page with travel packages
- 📍 Multiple destination options
- 📅 Trip booking form
- 🔐 Login and Registration pages
